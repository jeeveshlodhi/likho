## What is Likho?

Likho is a cross-platform note-taking workspace that combines rich block editing, Kanban boards, visual canvases, and AI assistance — all with 50+ pre-built templates so you start working, not setting up.

### Key capabilities

- **Notes** — Rich block editor with headings, toggles, checklists, code blocks, and wikilinks
- **Kanban** — Drag-and-drop task boards with labels, priorities, and custom columns
- **Canvas** — Freeform whiteboard for visual thinking and mind mapping
- **AI** — Local AI (runs offline on your device) plus cloud AI powered by Claude for smart writing, tagging, meeting intelligence, and more
