## 1. Create an account

Go to [likho.app](https://likho.app) and click **Get started — it's free**. You can also continue as a guest on the desktop app for a fully local experience.

## 2. Choose your workspace type

During onboarding you'll choose a workspace intent:

- **Personal** — notes, journal, second brain
- **Work** — projects, meetings, team collaboration
- **Study** — research notes, flashcards, knowledge base

You can change this later in **Settings → Workspace**.

## 3. Pick a template

Likho ships with 50+ templates. Click **+ New page** and choose from the template gallery, or start blank.

Popular starter templates:
- Daily Journal
- Meeting Notes
- Project Board (Kanban)
- Second Brain

## 4. Write your first note

The editor supports:

- `/` commands — type `/` anywhere to insert a block
- `[[wikilinks]]` — link to other notes with double brackets
- `#tags` — tag blocks inline
- **AI writing** — click the ✦ button or press `Cmd+Shift+A` to summon the AI assistant

## 5. Enable AI

- **Cloud AI** — sign in and go to **Settings → AI** to enable Claude-powered features
- **Local AI** — download the desktop app; local models run entirely on your device, no internet needed

## Keyboard shortcuts

| Action | Shortcut |
| --- | --- |
| New note | `Cmd+N` |
| Quick capture | `Cmd+Shift+N` |
| Global search | `Cmd+K` |
| AI writing assist | `Cmd+Shift+A` |
| Toggle sidebar | `Cmd+\` |
