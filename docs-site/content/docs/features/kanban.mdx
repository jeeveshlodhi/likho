## Overview

Likho Kanban boards give you a drag-and-drop workflow view. Create a new page and set its type to **Kanban** to get started.

## Columns

Boards come with three default columns:

- **Backlog** — ideas and future tasks
- **In Progress** — active work
- **Done** — completed items

Add, rename, or delete columns from the **⋯** menu on each column header.

## Cards

Each card supports:

- **Title** — the main task name
- **Description** — rich block editor inside the card detail view
- **Priority** — Urgent, High, Medium, Low
- **Labels** — color-coded tags
- **Due date**
- **Assignee** (Team plan)

Click a card to open the detail panel. All card content is full-featured BlockNote.

## Drag and drop

Drag cards between columns or reorder within a column. Changes sync in real time to all collaborators.

## Filters

Use the filter bar at the top to filter cards by:

- Label
- Priority
- Assignee
- Due date range

## Keyboard shortcuts

| Action | Shortcut |
| --- | --- |
| New card | `N` |
| Move card left | `Shift+←` |
| Move card right | `Shift+→` |
| Open card detail | `Enter` |
| Close card | `Esc` |
